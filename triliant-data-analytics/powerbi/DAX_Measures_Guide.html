Triliant Data — Power BI Dashboard Setup Guide

Analyst: Md. Abul Bashar Nirob | Company: Triliant Data (triliantdata.com) Period: March 2025 – February 2026 | Offices: Dhaka & Chittagong, Bangladesh

Step 1 — Import the Excel File

  1. Open Power BI Desktop
  2. Click Home → Get Data → Excel Workbook
  3. Select Triliant_Data_Analytics.xlsx
  4. Check all sheets: Raw Data, KPI Summary, Service Analysis, Client Tracker
  5. Click Load

Step 2 — Data Model Relationships

In Model View, create these relationships:

From Table Column To Table Column Cardinality
Raw Data Month KPI Summary Month Many-to-One
Raw Data Client Client Tracker Client Many-to-One
Raw Data Service Type Service Analysis Service Type Many-to-One

Step 3 — DAX Measures (copy into Power BI)

Paste each measure into Home → New Measure:

-- ── REVENUE MEASURES ─────────────────────────────────────

Total Revenue =
SUM('Raw Data'[Revenue (BDT)])

Revenue (BDT M) =
DIVIDE([Total Revenue], 1000000)

MoM Revenue Growth =
VAR CurrentRev = [Total Revenue]
VAR PrevRev =
    CALCULATE(
        [Total Revenue],
        DATEADD('Raw Data'[Month], -1, MONTH)
    )
RETURN
    DIVIDE(CurrentRev - PrevRev, PrevRev)

YTD Revenue =
TOTALYTD([Total Revenue], 'Raw Data'[Month])

Revenue Target =
[Total Revenue] * 1.10

Revenue vs Target =
DIVIDE([Total Revenue], [Revenue Target]) - 1

-- ── OPERATIONAL MEASURES ─────────────────────────────────

Avg Report Time =
AVERAGE('Raw Data'[Report Time (hrs)])

Report Time Reduction % =
VAR FirstMonth = CALCULATE(
    AVERAGE('Raw Data'[Report Time (hrs)]),
    TOPN(1, ALL('Raw Data'[Month]), 'Raw Data'[Month], ASC)
)
VAR LastMonth = CALCULATE(
    AVERAGE('Raw Data'[Report Time (hrs)]),
    TOPN(1, ALL('Raw Data'[Month]), 'Raw Data'[Month], DESC)
)
RETURN
    DIVIDE(FirstMonth - LastMonth, FirstMonth)

Avg Satisfaction =
AVERAGE('Raw Data'[Satisfaction (1-5)])

Avg KPI Achievement =
AVERAGE('Raw Data'[KPI Achievement (%)])

-- ── LEAD & CONVERSION MEASURES ───────────────────────────

Total Leads =
SUM('Raw Data'[New Leads])

Total Conversions =
SUM('Raw Data'[Conversions])

Conversion Rate =
DIVIDE([Total Conversions], [Total Leads])

-- ── CLIENT MEASURES ──────────────────────────────────────

Active Clients =
CALCULATE(
    DISTINCTCOUNT('Raw Data'[Client]),
    'Raw Data'[Project Status] = "Active"
)

Clients by Region =
CALCULATE(
    DISTINCTCOUNT('Raw Data'[Client]),
    ALLEXCEPT('Raw Data', 'Raw Data'[Region])
)

Revenue per Client =
DIVIDE([Total Revenue], DISTINCTCOUNT('Raw Data'[Client]))

-- ── ON-TIME DELIVERY ────────────────────────────────────

On-Time Delivery Rate =
DIVIDE(
    CALCULATE(COUNTROWS('Raw Data'), 'Raw Data'[On-Time] = "Yes"),
    COUNTROWS('Raw Data')
)

-- ── SERVICE ANALYSIS ────────────────────────────────────

Revenue Share by Service =
DIVIDE(
    [Total Revenue],
    CALCULATE([Total Revenue], ALL('Raw Data'[Service Type]))
)

Top Service =
TOPN(1, VALUES('Raw Data'[Service Type]), [Total Revenue], DESC)
